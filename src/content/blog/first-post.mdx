Welcome to my new blog! This space is built with **Astro** and **MDX**, giving me the power of static site generation combined with the flexibility of JSX components inside markdown.

## What to Expect

Here you'll find posts about:

- Web development best practices
- Astro framework tips and tricks
- Modern CSS techniques
- Thoughts on developer experience

## Why Astro?

Astro is a fantastic framework for content-driven sites because it ships zero JavaScript by default. That means fast page loads and great performance scores right out of the box.

> "The best code is no code at all." — Every performance engineer ever
